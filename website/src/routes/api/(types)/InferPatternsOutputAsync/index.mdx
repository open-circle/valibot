# InferPatternsOutputAsync

Infer patterns output async type.

> This type is too complex to display. Please refer to the [source code](https://github.com/open-circle/valibot/blob/main/library/src/schemas/recordWithPatterns/recordWithPatternsAsync.ts).

```ts
// Create async-compatible pattern tuples
const patterns = [
  [v.picklist(['id']), v.pipe(v.string(), v.transform(Number))],
  [v.picklist(['name']), v.string()],
] as const;

// Infer patterns output async type
type PatternsOutputAsync = v.InferPatternsOutputAsync<typeof patterns>; // { id?: number } & { name?: string }
```
