# Valibot AI Agent Prompts

Quick reference for selecting the right prompt guide.

## Decision Tree

| Task                                                  | Guide                                                        |
| ----------------------------------------------------- | ------------------------------------------------------------ |
| Navigate repository, find files, understand structure | [repository-structure.md](./repository-structure.md)         |
| Write/fix JSDoc or inline comments in library source  | [document-source-code.md](./document-source-code.md)         |
| Add new API page to valibot.dev                       | [add-new-api-to-website.md](./add-new-api-to-website.md)     |
| Update existing API page after source changes         | [update-api-on-website.md](./update-api-on-website.md)       |
| Add tutorial/conceptual guide to website              | [add-new-guide-to-website.md](./add-new-guide-to-website.md) |

## Key Principle

**Source code is the single source of truth.** All documentation must match the actual implementation in `/library/src/`.
